# Logo Setup Instructions

Please add the SmartUp Academy logo file to this directory (`public/`).

**File name:** `smartup-logo.png`

The logo should be placed at: `/room-reserve/room-reserve/public/smartup-logo.png`

The logo will automatically appear in:
- Calendar page navigation
- Dashboard navigation  
- Admin page navigation

If the logo file is not found, the system will fallback to the default icon.
